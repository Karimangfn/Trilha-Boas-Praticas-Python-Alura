📚 Curso da plataforma Alura focado em estruturar melhor o código utilizando princípios de design e padrões de projeto em Python.

* 🔗 Curso na Alura: *https://www.alura.com.br/curso-online-python-construindo-sistemas-eficientes-solid-design-patterns*

---

## 🎯 Sobre o curso

Este curso tem como objetivo mostrar como organizar melhor o código aplicando os princípios do SOLID e alguns dos principais Design Patterns.

Ao longo das aulas, são trabalhados conceitos que ajudam a reduzir acoplamento, melhorar a organização e deixar o código mais preparado para mudanças, mantendo ele mais limpo e fácil de evoluir.
