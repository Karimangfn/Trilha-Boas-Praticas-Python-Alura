# 🧪 Python e TDD: explorando testes unitários (Alura)

📚 Curso da plataforma Alura focado em testes automatizados em Python utilizando Pytest e práticas de TDD.

---

## 🎯 Sobre o curso

Este curso tem como objetivo ensinar como garantir a qualidade do código através de testes automatizados em Python.

Ao longo das aulas, são apresentados conceitos de testes unitários, uso do Pytest e práticas de TDD, ajudando a escrever código mais seguro, confiável e fácil de manter ao longo do tempo.
